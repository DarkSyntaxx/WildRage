# WildRage

A fast-paced **2D pixel-art fighting game** built using **HTML, CSS, and JavaScript (Canvas API)**.
Inspired by classic arcade fighters, this project focuses on delivering responsive combat mechanics, smooth animations, and an engaging gameplay experience.

---

## 🎮 Features

* ⚔️ Real-time 1v1 fighting system
* 🧍 Multiple playable characters with unique abilities
* 🧠 AI opponent with adjustable difficulty
* ❤️ Health bars and round-based gameplay (Best of 3)
* 💥 Combo system with dynamic hit effects
* 🔥 Special moves with energy mechanics
* 🛡️ Blocking, parry, and counter systems
* 🎨 Pixel-art visuals with smooth animations
* 🌆 Multiple stages with immersive backgrounds
* 🎵 Sound effects and background music
* 🧪 Training mode for practice and testing
* 🏆 Leaderboard and achievements system
* 🛍️ Skins and in-game currency system

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript (ES6)**
* **Canvas API**

---

## 🧠 Learning Highlights

This project demonstrates:

* Game loop implementation using `requestAnimationFrame`
* Collision detection and advanced hitbox systems
* State-driven animation handling
* Object-oriented design in JavaScript
* AI behavior and decision-making logic
* Performance optimization techniques
